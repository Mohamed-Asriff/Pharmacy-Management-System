Update Your Database

Step 1: Add Image Column (If you already created the database)

Open phpMyAdmin and run this SQL query:

ALTER TABLE medicines
ADD COLUMN image VARCHAR(255) DEFAULT 'default-medicine.svg' AFTER expiry_date;

Or import: sql/add_image_column.sql

Step 2: For New Database

If you haven't created the database yet, simply import:

sql/pharmacy.sql

This already includes the image column!

Add Custom Product Images

Step 1: Prepare Your Images

  • Format: JPG, PNG, or SVG
  • Recommended size: 400x400 pixels
  • Use descriptive names: paracetamol-500mg.jpg
  • Keep file size under 500KB for fast loading

Step 2: Upload Images

Copy your images to this folder:

c:\xampp\htdocs\POS - Copy\images\products\

Step 3: Update Database

In phpMyAdmin, update the medicine record:

UPDATE medicines
SET image = 'paracetamol-500mg.jpg'
WHERE sku = 'PAR500';

Bulk Update Images

Pro Tip

You can update multiple medicines at once using category-based logic or by importing a CSV file with medicine SKU and image filename pairs.

-- Update all diabetes medicines
UPDATE medicines m
JOIN categories c ON m.category_id = c.id
SET m.image = 'insulin-injection.jpg'
WHERE c.name = 'Diabetes' AND m.name LIKE '%Insulin%';

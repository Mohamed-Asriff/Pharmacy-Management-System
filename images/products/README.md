# Product Images Folder

## Upload Instructions

Place your product images in this folder with the following naming convention:

- Use lowercase letters
- Replace spaces with hyphens
- Use .jpg, .png, or .webp format
- Recommended size: 400x400 pixels

## Examples:
- `paracetamol-500mg.jpg`
- `cetrimide-cream.jpg`
- `insulin-glargine.jpg`

## Default Image
If a medicine doesn't have a specific image, `default-medicine.jpg` will be shown.

## Sample Images
You can download free medicine/pharmacy stock images from:
- Unsplash.com
- Pexels.com
- Pixabay.com
